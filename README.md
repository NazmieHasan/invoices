# invoices

Bootstrap v3.3.7

PHP 7.2.34, MySQL

Symfony 3.4.47

composer update

php bin/console doctrine:database:create

php bin/console doctrine:schema:update

php bin/console doctrine:schema:update --dump-sql

php bin/console doctrine:schema:update --force

php bin/console server:run
